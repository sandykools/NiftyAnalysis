# Testing Folder Structure

This folder contains all test and debug scripts organized by category.

## Folder Structure:
- **debug/** - Debugging scripts for specific issues
- **tests/** - Comprehensive test scripts
- **checks/** - Verification and validation scripts  
- **fixes/** - Fix scripts for various issues
- **api_tests/** - API testing scripts
- **misc/** - Miscellaneous files

## How to Run:
Most scripts can be run with: python script_name.py

## Notes:
- These are diagnostic scripts, not part of the main application
- Some scripts require authentication or specific setup
- Keep this folder organized for future debugging
